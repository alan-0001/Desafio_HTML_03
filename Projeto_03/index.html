CX-One

O software CX-One permite aos utilizadores criar, configurar e programar um conjunto de dispositivos, como, por exemplo, PLCs, HMIs, bem como redes e sistemas de controlo de movimento utilizando apenas um pacote de software com um número de licença e instalação. Deste modo, a complexidade da configuração é significativamente reduzida permitindo a programação ou configuração de sistemas de automatização com uma formação mínima.
